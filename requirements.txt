=========================================
팀 프로젝트 자동 조 편성 시스템 실행 안내
=========================================

[1] 파일 구성 확인
프로젝트 폴더 내에 다음 파일들이 모두 모여 있는지 확인해 주세요.
1. app.py (메인 파이썬 실행 코드)
2. students_db.csv (학생 명단 및 학번/이메일이 포함된 원본 데이터)
3. Plaintext.txt (현재 안내 파일)

[2] 필수 라이브러리 설치
프로그램을 실행하려면 아래의 패키지들이 필요합니다. 
VS Code의 터미널(Terminal) 창을 열고 아래 명령어를 그대로 복사하여 실행해 주세요.

pip install streamlit pandas openpyxl


[3] 프로그램 실행 방법
라이브러리 설치가 완료되었다면, 터미널 창에 아래 명령어를 입력하여 웹사이트를 실행합니다.

streamlit run app.py

(명령어를 입력하면 기본 웹 브라우저가 자동으로 열리며 시스템이 구동됩니다.)


[4] 관리자(Admin) 접속 정보
학생들의 응답을 엑셀로 다운로드하거나, 자동 조 편성 및 초기화를 진행하려면 아래의 계정으로 로그인하세요. (필요시 app.py 코드 내에서 수정 가능합니다.)
- ID (이메일 칸에 입력) : admin
- Password (학번 칸에 입력) : admin


[5] 웹 배포(Deploy) 시 주의사항
추후 Streamlit Community Cloud 등을 통해 웹에 배포하여 학생들에게 링크를 공유하실 때에는, 이 폴더 안에 반드시 `requirements.txt` 라는 이름의 파일이 필요합니다. 
해당 파일을 새로 만들고 아래 세 줄만 적어서 저장한 뒤 같이 업로드해 주세요.

streamlit
pandas
openpyxl
